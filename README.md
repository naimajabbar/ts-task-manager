## TypeScript CLI Task Manager

This is a simple Command Line Interface (CLI) task manager built using **TypeScript** and **Node.js**. The project demonstrates how to structure a TypeScript application sing modules, classes, and services while compiling TypeScript into JavaScript.

The application allows users to manage tasks directly from the terminal. Users can add new tasks, view all tasks, update existing tasks, and delete tasks using CLI commands.

### Features

* Add a new task with id, title, description, and status
* List all tasks from the task manager
* Update task details such as title, description, or status
* Delete tasks by id
* Modular project structure using **models**, **services**, and **CLI controller**

### Technologies Used

* TypeScript
* Node.js
* ES Modules
* Command Line Arguments (`process.argv`)

### Purpose

The purpose of this project is to practice:

* TypeScript types and interfaces
* Object-oriented programming with classes
* Modular project structure
* CLI argument handling in Node.js
* TypeScript compilation using `tsc`
